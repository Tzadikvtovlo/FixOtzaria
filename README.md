# FixOtzaria
תיקון קובץ manifest של Otzaria

נועד למקרה של ספרים חסרים בספריית אוצריא, ועקב קובץ manifest שגוי התוכנה אינה מייבאת אותם מן המאגר.

# הוראות הפעלה:
הורדת קובץ bat של הגירסא האחרונה, והפעלתו.

לאשר הרשאת מנהל, ולחכות לסיום ההרצה.

אם ספריית אוצריא אינה מותקנת במיקום ברירת המחדל (C>אוצריא), תתבקשו להזין את הנתיב של הספריה.

בסיום ההרצה, הסקיפט יעדכן כמה ספרים נמצאו חסרים.

ההודעה הינה באנגלית ובעברית, הכיתוב בעברית מופיע כג'יבריש, אך ניתן ע"י העתקה להדביק אותו בדפדפן או בוורד ויוצג באופן תקין.

תיקון קובץ המניפסט אינו מוסיף את הספרים, לשם כך נצרך להפעיל את אוצריא, ואז הסנכרון האוטומטי מייבא מחדש רק את הספרים שנמצאו חסרים.
